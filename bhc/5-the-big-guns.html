The Big Guns

There's more than one way to use CSS, and it can get pretty fancy
In this lesson we're going to implement a common pattern: a centered fixed-width page

1. Add a container

The first thing we're going to do is add a container around our entire page's content. We're going to use a div element, which is used to define a section in the document. For example, here is a before and after:

Before

<!doctype html>
<html>
  <head>
    <title>Hello</title>
    <style>p { color: #ff0000; }</style>
  </head>
  <body>
    <h1>Ice Cream</h1>
    <p>I love ice cream!</p>
  </body>
</html>

After

<!doctype html>
<html>
  <head>
    <title>Hello</title>
    <style>p { color: #ff0000; }</style>
  </head>
  <body>
    <div>
      <h1>Ice Cream</h1>
      <p>I love ice cream!</p>
    </div>
  </body>
</html>

Note that adding a div doesn't actually change the page at all.

2. Add a CSS class

If we added a rule for divs, it would apply to every div on the page. However, we only want to apply this rule to this one kind of div. To do this, we're going to add a CSS class to the div, like this:

<div class="container">

You only need to add the class to the opening tag, not the closing one.

3. Add the style

Now that our class is in place, we can reference the class in our CSS using .container. A . prefix means that we are targeting a CSS class name, and not a tag name. For example, let's simply change the background color:

.container {
  background-color: #cccccc;
}

Refresh the page, and you'll see that everything inside the container div has a gray background.

4. Fix the width

We'd like our page to have a fixed with, so let's set the width of our container div.

.container {
  background-color: #cccccc;
  width: 600px;
}

This will set the width of the div to exactly 600 pixels.

5. Center it

Centering is where we're going to get a little crafty. We're going to use the margin CSS rule, which will add space around an object. Instead of adding a fixed amount of space, we're going to tell the browser to automatically space it for us. The margin rule takes 4 values: top right bottom left. Let's set it up.

.container {
  background-color: #cccccc;
  width: 600px;
  margin: 0 auto 0 auto;
}

The top and bottom values are 0, since we don't want to add any extra space there. And the left and right margins are set to auto to let the browser automatically position it horizontally. Refresh the page and check it out.

6. Discussion

Try different values for the top and bottom margin, the width, and the background color. Once you've found something you like, share it with your neighbor.
