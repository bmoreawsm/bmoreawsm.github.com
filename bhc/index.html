Pre-requisites

Before starting the lessons, please do the following:

Sublime Text

There are many text editors available, from free to expensive and from simple to complex. I've chosen Sublime Text for this class as it is simple, but offers many features if you want to dig down into them.

Download Sublime Text »

Dropbox

Dropbox has a public folder feature which we will use to host our html files.

Sign up and install Dropbox »

Google Chrome

Google Chrome has a built-in html and css inspector which we will use during the course.

Download Google Chrome »
