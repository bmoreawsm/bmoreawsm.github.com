Make an HTML File

A basic web site is file-based. In this lesson, we'll make a simple html file.

1. Make a file

  1. Open Sublime Text
  2. Type "hello world"
  3. Save the file on your desktop as "hello.html"
  4. Open Google Chrome
  5. Click "File" then "Open File" and open "hello.html"

As you can see, your web browser can open plain text files.

2. Make it html

  1. Switch back to Sublime Text
  2. Delete the text in hello.html
  3. Copy and paste this code into the file: 
    <!doctype html>
<html>
  <head>
    <title>hello</title>
  </head>
  <body>
    <h1>hi</h1>
  </body>
</html>
