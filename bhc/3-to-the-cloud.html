To the Cloud!

Web site are only fun if everyone can see them. In this lesson, we'll put our page online

1. Setup Dropbox

If you haven't already, setup Dropbox.

2. Make it public

Copy (or move) your html file to the "Public" folder within your Dropbox.

3. Share it

Get the public link for your file by right clicking the file and choosing "Copy Link" or "Public Link". Share this file with your neighbor, and have them view your page in their browser.
