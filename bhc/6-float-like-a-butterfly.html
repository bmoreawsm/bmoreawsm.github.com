Float Like a Butterfly

Horizontally laying out content is done with divs and floating
In this lesson we're going to implement another common pattern: a sidebar

1. Add a content div and a sidebar div

Similar to the last section, we need to wrap a portion of our content inside a div. Wrap everything within the container div inside another div called "content". Also, before the content, make a sidebar div with our future sidebar content in it. So now, your page structure should look something like:

<div class='container'>
  <div class='sidebar'>
    This is my sidebar!
  </div>
  <div class='content'>
    Your page content
  </div>
</div>

What you'll notice is that the sidebar is on top of the content. Next, we'll move it to the side with css.

2. Float the sidebar and content with css

Floating tells an element to flow horizontally in a particular direction. In our case, we want the sidebar and content to flow to the left (left-aligned). We need to do two things: set their float property and give them a smaller width so they don't take up all the space. For the width, we need to be sure that their total width is less than or equal to the container width so they can fit next to each other.

.container {
  background-color: #cccccc;
  width: 600px;
  margin: 0 auto 0 auto;
}

.sidebar {
  float: left;
  width: 200px;
}

.content {
  float: left;
  width: 400px;
}

Now, the sidebar should be next to the content!

6. Discussion

Experiment with different divisions of your available width to divide up the page in different ways. Think about this: can every page of the web be created simply with horizontal floats and vertical divs?
