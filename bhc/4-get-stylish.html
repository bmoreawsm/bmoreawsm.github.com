Get Stylish

The web wouldn't be very fun if everyone's site looked the same
In this lesson we'll add some personality to our pages with CSS

1. Add a Style Tag

CSS is added to a page with a style tag. Add a style tag within your page's head tag, like this:

<head>
  <title>Hello</title>
  <style></style>
</head>

Right now it's empty, let's put something in there.

2. Add some color

Let's make our paragraphs more colorful. Inside the style tag, add this:

p {
  color: #ff0000;
}

The p says that all the styles within the { and } apply to p tags. The color rule is used to change the color of the text. The color rule accepts a css color as a value, in this case we're using hex notation for red. Every line with a rule must end with a ;.

3. Discussion

Take a look at W3School's CSS reference for text. Pick one of these rules, learn about it, and add it to your page. Share the rule and what it does with the rest of the class.
