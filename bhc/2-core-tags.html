Core Tags

The HTML language consists of a wide vocabulary of tags
In this lesson, we'll cover the essential ones

1. Headings

We've already seen a heading, it's the h1 in our site. H is for heading and the 1 denotes the priority (1 is the highest).

  1. Change the text inside the h1 to a topic that interests you
  2. Add an h2 tag below it with the text of a subtopic
  3. Add an h3 below that with another subtopic
  4. Refresh your browser to see how it renders headings

2. Text

The most basic block of text is the p tag. p stands for "paragraph".

  1. Beneath your h3, add a p tag and type in a few sentences about that topic
  2. Refresh your browser and check it out
  3. Now add another p tag below the first one and type another few sentences
  4. Refresh your browser, and notice how it automatically puts some vertical space between the p tags

3. Links

Links are tags that have a property: the url they point to. These properties are inside the opening tag. For example, here is a link to Google:

<a href="http://google.com">Google</a>

The href is the url that this link should go to. Properties are written in the property="value" format.

Add a link to your page.

4. Lists

Lists are the first compound tag we're learning about. There are two kinds of lists, ol (ordered lists) and ul (unordered lists). Lists have sub-tags li (list item). For example, here is one of each:

<ol>
  <li>One</li>
  <li>Two</li>
  <li>Three</li>
</ol>

<ul>
  <li>Alpha</li>
  <li>Beta</li>
  <li>Gamma</li>
</ul>

Notice how the list items tags are within the list tags. That is so the list knows when to start and stop it's numbering.

Add an ordered list or an unordered list to your page, your choice. Fill it with some items that make up your topic. Or maybe just your favorite kinds of ice cream.

5. Discussion

W3Schools is an invaluable resource for html. On the left of the page, check out the section "HTML5 Tags". Pick a tag out of that list, and explain how it works. We'll each take a turn.
