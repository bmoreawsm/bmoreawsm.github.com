Making Things Fit

One of the most challenging aspects in html & css is laying things out
In this lesson we'll try a few different layout styles

1. A Simple form

Forms are a great example of combining a variety of objects in complicated layouts. For this example, we'll build a survey and try a few different layout techniques.

Let's start with our form, in a new file with its own stylesheet called form.html:

<form>
  <fieldset>
    <legend>Contact Info</legend>
    
    <label for='name'>Name</label>
    <input name='name' id='name'>

    <label for='email'>Email</label>
    <input name='email' id='email'>
  </fieldset>

  <fieldset>
    <legend>Feedback</legend>

    <input type='checkbox' name='satisfied' id='satisfied'>
    <label for='satisfied'>I was satisfied</label>

    <p>How satisfied were you?</p>

    <input type='radio' name='level' value='very' id='level-very'>
    <label for='level-very'>Very</label>

    <input type='radio' name='level' value='not-much' id='level-not-much'>
    <label for='level-not-much'>Not much</label>

    <label for='comments'>Additional comments:</label>
    <textarea name='comments'></textarea>
  </fieldset>
</form>

Take a look. Pretty terrible, isn't it? Aesthetics aside, there are some major layout issues. For example:

  1. Label and input widths are not consistent
  2. Fields are not aligned horizontally with each other
  3. Text area does not align vertically with its label

What else?

2. Grouping

First of all, let's group our form elements so that the label and input are grouped in a div together. This will give the form a vertical feel. For example:

<div class='field'>
  <label for='name'>Name</label>
  <input name='name' id='name'>
</div>

3. Width

Now, let's get a handle on the widths of elements, first the form:

form {
  width: 400px;
  margin: auto;
}

For the Information section, let's use a horizontal layout:

fieldset.horizontal input {
  display: block;
  width: 75%;
  float: right;
}
fieldset.horizontal label {
  display: block;
  width: 10%;
  clear: both;
  float: left;
}

On your own, fix up the vertical alignment of the name and email fields. Keep in mind an input's height is the combination of height, padding, and border. A label is text, so its height is based on line-height. Style the input elements a bit nicer by changing padding, border, and event font-size.

4. Spacing

The checkbox could use some room to breath, and so could the radio buttons. Let's add some space:

.field.checkbox {
  padding-left: 40px;
}

.field.checkbox input {
  margin-left: -30px;
  float: left;
}

.field.checkbox label {
  display: block;
}

We'll need to add the checkbox class to the field container for our checkbox. We could have put fixed-width containers around the input and the label to space them out, but this requires much less html markup. It also allows for different browsers to have differently sized checkbox objects and they all fit as long as they're less than 30 pixels wide.

5. Radio

Now let's work on the radio buttons. Perform the same style as the checkboxes, but in addition, lay them out horizontally, 50/50 like the name and email field.

6. Textarea

We'll end on an easy one. Have the textarea be full width, below its label. And give it a few more rows so there's space to type.

7. Tweak

Make some other improvements to the form. Aesthetic or layout. You're the designer!
