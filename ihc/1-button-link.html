Button Link

Many sites feature links that look like buttons
In this lesson, we'll make a utility class for button links

1. Setup an html scaffold

Create the following files in a new folder:

  1. index.html
  2. about.html
  3. styles.css

In index.html, create an html5 scaffold:

<!doctype html>
<html>
  <head>
    <title>Index</title>
    <link href="styles.css" type="text/css" rel="stylesheet">
  </head>
  <body>
    <h1>Index</h1>
  </body>
</html>

In about.html, place the same code, but change the header:

<!doctype html>
<html>
  <head>
    <title>Index</title>
    <link href="styles.css" type="text/css" rel="stylesheet">
  </head>
  <body>
    <h1>About</h1>
  </body>
</html>

In styles.css, make a small change to ensure it is working:

h1 {
  color: #f00;
}

Open index.html and about.html in your browser. Ensure index.html says Index and about.html says About. Ensure the headers are red in both cases.

2. Link between the sites

Let's add a link from one page to the other. Below the header in index.html, add the following link:

<a href='about.html'>About</a>

This is known as a relative link, because the link is relative to the location of the file it is in. Without a / at the front of the href, it will look in the current folder.

Click the link. It should take you to the about page. Add a link to the index on the about page.

3. Put about in a folder

Our site would get disorganized if we put everything at the root. Create a folder called "info" and move about to that folder.

Our links are now broken. Edit index.html and change the link to "info/about.html". Now edit about.html, what should the link be? Here's a hint: ".." means "the folder above me".

4. Basic button link style

Now that our html structure is all set up, let's make the link look nice. We'll start by making it look boxier:

.bma-button {
  padding: 4px 8px;
  border: 1px solid #eee;
  -webkit-border-radius: 5px;
          border-radius: 5px;
}

Let's ask a few questions:

  1. How do we add that style to our link?
  2. What is better, a css class generally named "button" or specifically named "about-link"? Why?
  3. Why do we need two rules for border radius?

5. Gradients

The button is pretty plain, let's modify the background to give it a more rounded feel:

.bma-button {
  background-image: -webkit-linear-gradient(top, #faa, #c88);
  background-image:    -moz-linear-gradient(top, #faa, #c88);
  background-image:      -o-linear-gradient(top, #faa, #c88);
  background-image:     -ms-linear-gradient(top, #faa, #c88);
  border: 1px solid #c88;
}

We also update the color and border color for continuity.

6. Hover transition

It looks great, until we move our mouse over it. Let's add a hover state:

.bma-button {
  background-color: #c88;
  background-repeat: repeat-x;
}
.bma-button:hover {
  background-position: 0 -10px;
  color: white;
  text-decoration: none;
}

We have to add in a background color and repeat so our hover simply slides the gradient up, which makes it look hovered.

But that's not enough for us, let's make it extra snazzy with a sliding transition:

.bma-button:hover {
  -webkit-transition: background-position 300ms ;
     -moz-transition: background-position 300ms ;
      -ms-transition: background-position 300ms ;
       -o-transition: background-position 300ms ;
          transition: background-position 300ms ;
}

Last, let's add an "active" state, which is used when the mouse button is down:

.bma-button:active {
  background-image: none;
  -webkit-box-shadow: inset 0 1px 2px rgba(0, 0, 0, 0.15);
     -moz-box-shadow: inset 0 1px 2px rgba(0, 0, 0, 0.15);
          box-shadow: inset 0 1px 2px rgba(0, 0, 0, 0.15);
}

Using an inset box shadow gives the impression of a button that is being pushed into the surface. Active states are very subtle are a noticable feedback for the user.

7. Colors

We're using the background-image and background-color to set the color of the buttons. Experiment with a few different colors schemes. Then, encapsulate these color schemes in an additional button class, so we can have a few different kinds of buttons. For example, I could use them like this:

<a href='#' class='bma-button bma-button-green'>Click me</a>
